# WebComponents

### 示例

| 组件名称       | 作用                                | 使用知识点                                               |
| -------------- | ----------------------------------- | -------------------------------------------------------- |
| popup-info     | 显示自定义的提示信息                | connectedCallback 生命周期，编程式 在影子DOM树中应用样式 |
| word-count     | 字数统计                            | connectedCallback，attributeChangedCallback 生命周期     |
| emoji-switch   | 表情 开关组件（重要的是实现细节）   | connectedCallback 生命周期，编程式 在影子DOM树中应用样式 |
| expanding-list | 优化的 列表组件（重要的是实现细节） | connectedCallback 生命周期，自定义继承元素 ul            |
| custom-square  | 自定义正方形                        | connectedCallback，attributeChangedCallback生命周期      |
| filled-circle  | 自定义填充圆形                      | 声明式 在影子DOM树中应用样式                             |
